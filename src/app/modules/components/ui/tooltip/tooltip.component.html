UI Tooltip

Exemplos para utilização do componente Tooltip.

Tooltip ou dica de contexto é um elemento comum de interface gráfica. Tooltip é aquela moldura flutuante que abre quando passa-se o mouse sobre um elemento da interface e que contém uma explicação adicional sobre tal elemento. É utilizado em conjunto com um cursor, normalmente um ponteiro do mouse.

Configuração

Após instalar o componente, será necessário importar o Modulo do Tooltip nos modulos da aplicação, conforme exemplo abaixo.

Texto

O texto que será exibido no tooltip deve ser definido no atributo "text"

Posição de abertura

Por padrão, o tooltip será aberto para cima. Essa posição pode ser alterada definindo o atributo "position".
