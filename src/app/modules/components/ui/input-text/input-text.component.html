UI Input Text

Exemplos para utilização do componente Input Text.

Use o componente Input Text para permitir o usuário inserir valores de texto, email, número, senha, URL, número de telefone e outros.

Configuração

Após instalar o componente, será necessário importar o Modulo do InputText nos modulos da aplicação, conforme exemplo abaixo.
